# NexusBoosters Permissions

- `nexusboosters.use` : Standard player permission required to open the GUI, check active boosters, and use boosters.
- `nexusboosters.give` : Admin permission allowing you to give booster tokens directly to player inventories.
- `nexusboosters.reload` : Admin permission to execute `/nb reload`.
- `nexusboosters.admin` : Admin permission to view backend administrative menus.
- `nexusboosters.booster.<id>` : (Optional) Specific permissions required to activate individual boosters if `permission: ` is defined in `boosters.yml`.
