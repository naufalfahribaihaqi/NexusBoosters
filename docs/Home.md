# Home

NexusBoosters is an All-in-One booster management plugin for PaperMC. It handles personal and global boosters, cross-server sync, and third-party integrations natively.

## Features

* Personal and Global scopes.
* MySQL Cross-Server synchronization.
* Paginated graphical user interfaces.
* Support for Vault, ShopGUI+, and PlayerPoints.

## Requirements

* Java 21 or higher.
* Paper 1.21.x or higher.

## Quick Installation

1. Download the `NexusBoosters.jar`.
2. Place it in the `plugins/` folder.
3. Restart the server.
