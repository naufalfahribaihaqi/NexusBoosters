# Quick Start

Get started quickly by giving your players a booster token to activate.

## Overview

A booster must first exist in `boosters.yml`. Once configured, an admin can give booster tokens to players. The player then activates it via the graphical menu.

## Usage

1. Create an entry in `boosters.yml` (e.g., `weekend_xp`).
2. Run `/nb give Steve weekend_xp 1 PERSONAL` to give a token.
3. Player types `/nb open` to view their tokens.
4. Player clicks the token in the GUI to activate it.
5. Player types `/nb active` to view their active multipliers.
