# Permissions

A list of all permission nodes required to access plugin features.

## Overview

All permissions default to `op`. You must assign them manually to your player groups via a permissions manager like LuckPerms.

## Permissions List

| Permission | Description | Default |
|---|---|---|
| `nexusboosters.use` | Allows opening the GUI and activating standard boosters. | `op` |
| `nexusboosters.give` | Allows giving booster tokens directly to players. | `op` |
| `nexusboosters.reload` | Allows hot-reloading configurations. | `op` |
| `nexusboosters.admin` | Grants access to admin commands and backend menus. | `op` |
| `nexusboosters.booster.<id>` | Specific permission to activate a designated booster if required. | `op` |

## Notes

> Assign `nexusboosters.use` to your default group to allow normal players to access the `/nb open` menu.
