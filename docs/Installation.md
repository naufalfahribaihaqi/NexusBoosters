# Installation

Follow these steps to safely install NexusBoosters on your server.

## Overview

NexusBoosters is a drag-and-drop plugin. It generates all necessary configuration files upon the first startup.

## Installation Steps

1. Verify your server runs Java 21 and Paper 1.21.x.
2. Download or compile the `NexusBoosters.jar`.
3. Stop the server completely.
4. Place the JAR file into the `plugins/` directory.
5. Install any required dependencies.
6. Start the server to generate default configurations.
7. Edit configurations, then restart the server.

## Dependencies

| Dependency | Required | Purpose |
|---|---:|---|
| Paper | Yes | Server platform |
| Vault | Optional | Economy support |
| PlaceholderAPI | Optional | Placeholder support |
| Floodgate | Optional | Bedrock player detection |
| PlayerPoints | Optional | Points boosters |
| ShopGUI+ | Optional | Shop boosters |
