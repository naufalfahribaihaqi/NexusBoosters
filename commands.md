# NexusBoosters Commands

| Command | Aliases | Permission | Description |
|---|---|---|---|
| `/nb help` | `?` | `nexusboosters.use` | Displays the help menu. |
| `/nb open` | `gui`, `menu` | `nexusboosters.use` | Opens the Booster GUI. Bedrock players get a text fallback. |
| `/nb active` | `list` | `nexusboosters.use` | View currently active boosters affecting you. |
| `/nb activate <booster>` | `use` | `nexusboosters.use` | Activate a booster by its ID. |
| `/nb give <player> <booster> <amount> [PERSONAL|GLOBAL] [duration]` | `add` | `nexusboosters.give` | Grants booster tokens to a specific player's inventory. |
| `/nb reload` | `rl` | `nexusboosters.reload` | Hot-reloads all configs and languages without wiping active timers. |
| `/nb startglobal <booster> [duration]` | `sg` | `nexusboosters.admin` | Start a global booster with optional custom duration. |
| `/nb stopglobal <booster>` | None | `nexusboosters.admin` | Stop a specific active global booster. |
| `/nb sync` | None | `nexusboosters.admin` | Force cross-server synchronization. |
| `/nb admin` | None | `nexusboosters.admin` | Access the admin controls menu (WIP). |
